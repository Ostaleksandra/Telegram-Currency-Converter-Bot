# Currency Converter Telegram Bot

Telegram-бот для конвертации валют.  
Бот принимает название валют и количество, после чего возвращает результат конвертации и текущий курс.

## Функциональность
Бот поддерживает следующие команды:

### /start или /help
Показывает инструкцию по использованию бота.

Формат команды:
<валюта_из> <валюта_в> <количество>

Пример:
доллар евро 100

---

### /values
Показывает список доступных валют для конвертации.

---

### Конвертация валют

Пользователь отправляет сообщение в формате:

доллар евро 100

Бот выполняет:

- проверку корректности введённых данных
- конвертацию валюты
- вывод результата и текущего курса

Пример ответа бота:

100 доллар = 92.5 евро  
Курс: 1 доллар = 0.925 евро

---

## Используемые технологии

- Python
- pyTelegramBotAPI (telebot)
- REST API для получения курсов валют
- обработка исключений

---

## Установка и запуск

1. Клонировать репозиторий

```
git clone https://github.com/your_username/currency-telegram-bot.git
```

2. Установить зависимости

```
pip install pyTelegramBotAPI
```

3. Добавить токен Telegram-бота в файл `config.py`

```
TOKEN = "ваш_токен"
```

4. Запустить бота

```
python bot.py
```

## Обработка ошибок

Бот обрабатывает:

- неправильный формат команды
- неизвестную валюту
- некорректное количество
- другие исключения
